                                  What is the Tech Radar?

                                  The Finaps Tools & Technologies Radar is a list of tools and technologies with an assessment result.

                                  Radar Structure

                                  The Radar tracks tools and technologies, called blips, organized into quadrants and rings. Quadrants represent types of blips, and rings indicate our usage recommendation.

                                  Blips

                                  Blips are technologies or techniques relevant to software development, often changing position in the Radar, indicating our confidence in them.

                                  Quadrants

                                  • Tools — Task-specific tools enhancing quality and efficiency.
                                  • Technologies — Skill-based tools requiring knowledge and experience.
                                  • Platforms — Foundations for software like cloud services or virtual platforms.
                                  • Methodologies & Practices — Approaches, processes, and best practices for software development and project management.

                                  Rings

                                  • Adopt — Proactively used in projects.
                                  • Trial — Undergoing evaluation.
                                  • Assess — Promising tools/technologies to monitor.
                                  • Hold — Not recommended for projects.

                                  Goal

                                  Ensure new tools/technologies are properly evaluated and the accepted list is up-to-date.
